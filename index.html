Unit 1 – basics of the web

Anything can exist on the web that you like

Major Pieces
• You

• Your computer running a web browser

• Internet to

• Servers via HTTP (protocol the browser uses to communicate with server)


HTML – hypertext markup language (main type of document on the web)
This glues everything together made up of: Text content (what you see), Markup (what it looks like),
References to other documents Links to other pages

Markup is made of tags
opening tag -- contents --/closing tag
This whole thing is called an element

b = bold
em = italics

HTML Attributes
tag attr = “value”-- contents --/closing tag
Attr= attributes (attributes are always a value) tags can have multiple attributes
Anchor tag (A tag) – for making links – the value is a website
a href=www.w3c.org/html -- contents -- /a

Images
img src=”url” alt=”text”
“text” is what displays if the image doesn’t load
Images do not have a closing tag b/c it is a void tag – has no content
Images appear in line with text

Whitespace
Forcing text to go to the next line br - void tag to get it to go to the next line
Inline (ending a line) – just has text
br em img
Multiple br inserts multiple line breaks p is not a void tag – will render as a paragraph

Block (makes an invisible box) – can have height and width

Container elements Span – inline
span class=”foo” -- text--/span
Div – block
div bar=”foo”--text--/div
